Data and R-code for a paper titled: "Penny Elimination and Inflation" by Oz Shy

Data source: https://www.atlantafed.org/banking-and-payments/consumer-payments/survey-and-diary-of-consumer-payment-choice

---- General Instructions ----

    Download the relevant 2 data (RDS) files and the R-code file into the same directory.

    Don't forget to reset the working directory in the R-code. If you are using RStudio, go to the Session menu and click on Set Working Directory.

    Run the code from the beginning to the end (don't skip any code in the middle because new variables may be defined).
